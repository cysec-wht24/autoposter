# Autoposter

A Spring Boot Telegram bot that queues and posts messages to social media platforms.

## Requirements

- Java 21
- Maven
- MySQL 8+
- Telegram Bot Token (from @BotFather)

## Setup

1. Clone the repo
2. Create the database:
```sql
   CREATE DATABASE autoposter;
```
